The following description is about the process for optimizing a process. It starts with confirming the basic information. After confirming the basic information, confirming if the process requires optimization needs to be done. After confirming if the process requires optimization, you need to look to see if the process can be simplified. Once looking to see if the process can be simplified occurs, you need to write the potential process improvements. After writing the potential process improvements, getting approval for processing improvements needs to be done. When getting approval for processing improvements is completed, defining what would make the improvements successful needs to be done. After defining what would make the improvements successful, communicating the process changes needs to be done. The process is now completed.
