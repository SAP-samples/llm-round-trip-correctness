The following description is about the client onboarding process for a marketing agency. It starts with having an onboarding call with the client. Once having an onboarding call with the client occurs, the next tasks should be creating first draft of the marketing plan , updating the project management profile and scheduling future calls, they should be done in parallel. After creating first draft of the marketing plan , you need to get approval for the first draft. When updating the project management profile is completed, researching the campaign ideas should be done. Once all pre-tasks are executed, then informing client of project start should be done. After informing client of project start, one or more of the following paths should be executed: making changes based on follow-up call, sending a post-onboarding survey. Once sending a post-onboarding survey occurs, having a follow-up call needs to be done. The process is now completed.
