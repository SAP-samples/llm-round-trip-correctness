The following description is about the project management process. It starts with making an introduction. After making an introduction, identifying the approval status should be considered. if all approved, then nothing needs to be done, otherwise you should identify the approval status. After that, the next tasks should be laying out a timeline and drawing up a project plan, they should be done in parallel. Once all pre-tasks are executed, then having a kickoff meeting needs to be done. After having a kickoff meeting, the next tasks should be keeping communication and recording KPIs, they should be done in parallel. Once all pre-tasks are executed, then evaluating project outcomes needs to be done. The process is now completed.
