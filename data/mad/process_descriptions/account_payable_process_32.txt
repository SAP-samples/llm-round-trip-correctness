The following description is about the account payable process. It starts with sending to the senior accountant in your department. When sending to the senior accountant in your department is completed, creating a receiving report should be done. Once creating a receiving report occurs, one or more of the following paths should be executed: reporting errors to relevant team member, confirming all documents have been amended. When confirming all documents have been amended is completed, doubling check a three-way match should be done. Afterwards, one or more of the following paths should be executed: storing in paid invoice file, creating voucher for the documents. Once storing in paid invoice file occurs, one or more of the following paths should be executed: entering the invoice into the accounts payable account, stamping all documents. The process is now completed.
