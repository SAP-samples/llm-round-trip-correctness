The following description is about the customer support process for the ticket management. It begins when you check the ticket type. After checking the ticket type, the next tasks should be determining ticket type as 'Simple Question' and determining ticket type as 'Technical Issue', they should be done in parallel. When determining ticket type as 'Technical Issue' is completed, determining ticket type as 'Billing Question' should be done. Once all pre-tasks are executed, then the next tasks should be testing technical issues and responsing ticket question, they should be done in parallel. Once responsing ticket question occurs, replicating the issue should be taken into account. if all could replicate, then nothing needs to be done, otherwise you should replicate the issue. The process is now completed.
