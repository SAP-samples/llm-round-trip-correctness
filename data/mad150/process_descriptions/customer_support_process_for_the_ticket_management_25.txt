The following description is about the customer support process for the ticket management. It begins when you check the ticket type. When checking the ticket type is completed, the next tasks should be determining ticket type as 'Technical Issue', determining ticket type as 'Simple Question' and determining ticket type as 'Marketing/PR Question', they should be done in parallel. Once all pre-tasks are executed, then the next tasks should be replicating the issue and responsing ticket question, they should be done in parallel. The process is now completed.
